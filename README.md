# Tweaked Mines

Team mode for **Super Battle Golf**.

## Features
- Assign players to teams (up to 4 teams)
- Shared team score between teammates
- Friendly fire toggle
- Player skin color changes based on team
- Player name tag color changes based on team
- Team scoreboard with total score per team, sorted by best score
- Compatible with French and English

## Installation
Place this package into the game folder so the DLL ends up at `BepInEx/plugins/Team_Mode/Team_Mode.dll`.
